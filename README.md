# Flight Booking App (MERN)

## Folder structure (open this folder in VS Code)

```
Flight-Booking-App-MERN/
  client/   # React frontend
  server/   # Node/Express backend
  README.md
```

## Run locally

### 1) Backend

In a terminal:

```bash
cd server
npm install
npm run dev
```

Backend runs on `http://localhost:6001` by default (configurable via `server/.env`).

### 2) Frontend

Open a second terminal:

```bash
cd client
npm install
npm start
```

Frontend runs on `http://localhost:3000`.
